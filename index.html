Industrial Iot Cloud
Demo
At Avantech Inc. I was tasked with creating an iot cloud platform that would be a suitable replacement for most SCADA applications and lend itself well to industrial applications. Built on Amazon Web Services it uses a combination of client and cloud software to emulate many of the alarming, action and reaction components of SCADA systems. It includes an online dashboard application that integrates into the platform for real time data updates.

Github / Demo

WebSocket Chat

ARM Arkinods
Github
A project for a university video games course written in pure arm v6 assembly on a raspberry pi written to gain insight into how hardware components such as the gpu, memory and interrupt registers interact. It is a full fledged copy of Brick breakers which includes different levels and power ups. All code for this project is custom with no external libraries to aid development.
